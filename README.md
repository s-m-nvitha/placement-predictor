# 🎯 Placement Predictor

A Machine Learning web application that predicts whether a student is likely to get placed based on their **CGPA** and **IQ**.

Built using **Flask**, **Logistic Regression**, and **Scikit-learn**.

---

## 📌 Project Overview

Placement Predictor is a simple Machine Learning project that predicts a student's placement chances.

The model takes two inputs:

- CGPA
- IQ

It uses a trained **Logistic Regression** model to classify whether a student is likely to be placed or not.

The web interface is built using **Flask** and provides an interactive user experience with a modern glassmorphism design.

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/placement-predictor.git
```

### 2. Navigate to the project folder

```bash
cd placement-predictor
```

### 3. Install the required packages

```bash
pip install -r requirements.txt
```

### 4. Run the Flask application

```bash
python app.py
```

### 5. Open your browser

Visit:

```text
http://127.0.0.1:5000
```
